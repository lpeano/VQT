## 6. Fix Tecnici Applicati in Questa Sessione

### Bug HDF5 — topological_validation mancante nei file resumed

**Causa:** La scrittura del gruppo `topological_validation` avveniva in FASE 5, dopo il loop di simulazione. Un kill SIGTERM/SIGKILL lasciava il file senza quel gruppo.

**Fix strutturale (`generate_topological_dataset.py`):** Il gruppo viene ora creato con dataset HDF5 estendibili (`maxshape=(None,)`) prima dell'inizio del loop. Ogni step scrive e fa flush a blocchi di 10, rendendo i dati persistenti anche in caso di interruzione forzata.

**Fix post-hoc (`experiments/merge_l3_runs.py`):** Script che ricostituisce `topological_validation` da frame data (`positions`, `tau_locale`, `contorsione_locale`) applicando le formule del validator (f_closure, f_detorsion, ρ = 0.5·f_c + 0.5·f_d). Usato per recuperare `cosmo_L3_ext.h5`.
